About

  My name is Jeremy. I came to this industry via the electrical trade. My hope is to produce tools for users which decrease unnecessary frictions that can arise as humans begin to interact more and more with digital technology.

  I sincerely care about the craft of system and user interface design. A well made piece of software offers future adaptability and modifiability. I am most proud of my Capstone News Analyzer project (see below), which uses a combination of machine learning and dataset visualization tools to help the user gain a better intuition of how to ferret out misinformation. This web application implements a simple, minimal design in order to ensure that the app functionality is front and center.
